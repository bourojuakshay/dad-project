🔒 Auto Session Lock Test

How to Test:

  • Normal Operation: Move mouse, type, scroll, or click to reset the timer
  • Auto Lock: Stay inactive for 2 minutes to trigger automatic lock
  • Manual Lock: Click "Force Lock" to manually trigger the lock screen
  • Test PIN: Default PIN is "1234" (4 digits)
  • Failed Attempts: Enter wrong PIN 3 times to trigger lockout
  • Lockout: After 3 failed attempts, system locks for 30 seconds

📋 Test Scenarios

✅ Scenario 1: Normal Usage

1. Move mouse around

2. Type in inputs

3. Click buttons

4. Scroll the area

Expected: Timer resets, no lock occurs

🚨 Scenario 2: Inactivity Lock

1. Do nothing for 2 minutes

2. Or click "Force Lock"

3. Enter correct PIN (1234)

Expected: Lock screen appears, then unlocks with correct PIN

⚠️ Scenario 3: Failed Attempts

1. Trigger lock screen

2. Enter wrong PIN 3 times

3. Wait for lockout period

Expected: System locks for 30 seconds after 3 failed attempts

🔒 Scenario 4: Security Features

1. Try to paste in PIN field

2. Try right-click context menu

3. Try browser back/forward

Expected: Paste and context menu blocked, navigation restricted
